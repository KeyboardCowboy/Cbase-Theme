cBase Drupal Base Theme
================================================================================
https://github.com/KeyboardCowboy5/Cbase-Theme

Features:
- Improves Appearnce UI with vertical tabs and theme ancestry.
- Provides additional theming preprocess hooks and variables through ancestry.
- Allows system stylesheets to be turned off.
- Provides other CSS snippets to enhance default appears for messages and
  unpublished content.
- Provides CSS to assist Admin, Admin Menu and Contextual modules work together.
